The JavaFX Script™ plugin for the NetBeans IDE provides features for programming in the JavaFX Script language. The JavaFX Script programming language is declarative and statically typed. It has the same code structuring, reuse, and encapsulation features that make it possible to create and maintain very large programs using the Java technology. It makes intensive use of the Java2D Swing GUI components and enables you to easily create graphical user interfaces or GUIs. Support for the JavaFX Script language in NetBeans IDE has the following features:

Advanced Source Code Editor

The language-aware editor indents and completes your source code written in the JavaFX Script language. The editor also performs semantic syntax highlighting, live parsing of your code, matches words and brackets, marks errors and occurrences, and has code folding capability. It has a toolbar that includes controls to enable or disable the preview feature. For more information, see Editing JavaFX Script Source Files.

OpenJFX Compiler

The OpenJFX™ Compiler is integrated with the JavaFX Script plugin and is used to compile JavaFX Script source code. Java source code that is included in the JavaFX projects are compiled with the Java compiler. The results of the compilation are placed into one JAR file, which can be used for deployment or distribution. For more information, see About Building JavaFX Script Applications.

Preview Feature

The JavaFX Script plugin includes a preview feature that allows you to view the effects of the changes you make to your source code without having to build and run your application each time you make any changes. If there are no compilation errors encountered and the source file represents UI components, Shape, or Frame classes, changes you make in the source editor will be displayed in the canvas pane located just above the source editor pane. For more information, see Using JavaFX Script Preview Feature.

Debugger

Currently, the debugger feature included with the JavaFX Script plugin only supports debugging of Java source code that is included in a JavaFX project. For more information, see Debugging JavaFX Script Applications.

Profiler

The Profiler feature is available at the project-level for JavaFX projects. For more information, see Profiling JavaFX Script Applications.
