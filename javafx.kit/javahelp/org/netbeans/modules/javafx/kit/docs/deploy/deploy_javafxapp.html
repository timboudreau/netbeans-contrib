Deploying JavaFX Script Applications

See Also 

You can deploy your JavaFX Script application as a standalone application or as an application enabled with Java Web Start. To deploy with the Java Web Start, you must first create a special launch file (JNLP file) for your application written in the JavaFX Script language. When you click on a link to that JNLP file, it causes the browser to launch Java Web Start, which then automatically downloads, caches, and runs the given JavaFX Script application.

To run a standalone application from the IDE:

  1. In the Projects window, select the main node for your project and right-click it.
  2. Select Run Project.

To run a standalone application from the command line:

  1. Navigate to the project's dist folder.
  2. Type the following at the command prompt:
           java -jar application_jar_name.jar  main_JavaFX_class_name.Main

To build and run a JNLP application using the IDE:

  1. In the Projects window, right-click your project's node and select Properties.
  2. In Project Properties dialog, select the Application > WebStart category.
  3. Select the Enable Web Start and Compress JARs with Pack 200 checkboxes on the right-hand pane.
  4. In the Categories pane, select the Run cateogry and select the Enable Run with Java Web Start checkbox.
  5. Click OK.
  6. In the Projects window, right-click the project's node again and select Build Project.
    The IDE adds the following files to your project's folders:
    • jnlp-impl.xml in nbproject folder - contains the set of ANT targets needed to generate the JNLP file, and all other related files.
    • master.jnlp file in your project's main folder - used to create the launhc.jnlp file.
    • launch.jnlp and launch.html files in the dist folder - the launch.html file is linked to launch.jnlp file)
  7. Right-click your project's node and select Run Project.

To run a standalone application with Java Web Start:

Do one of the following after you build the JNLP application for your JavaFX project:
  • Navigate to the project's dist folder and double-click the launch.jnlp file.
  • Navigate to the project's dist folder and type the following at the command prompt:
           javaws launch.jnlp
