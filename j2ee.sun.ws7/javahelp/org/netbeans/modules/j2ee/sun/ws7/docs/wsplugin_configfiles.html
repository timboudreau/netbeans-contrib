Configuration Files for Web Applications Targeted for Sun Java System Web Server

See Also

Sun JavaTM System Web Server 7.0 supports J2EE 1.4 standards for web applications.

In addition to the web.xml file, which is a J2EE 1.4 standard, another file, sun-web.xml, is needed for each web application that is deployed on Sun Java System Web Server 7.0. See Configuring Web Application Deployment Descriptors for more information about these files.
