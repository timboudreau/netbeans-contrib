Enabling Dynamic Verification

See Also

Note : This section is applicable only if your project is using Sun Java System Application Server 8.x as runtime.

To enable dynamic verification you must start the instrumentation of the application server instance before you deploy your project.

To enable dynamic verification:

  1. In the Projects window, select the project node, right-click and choose Verify Project.
    The Choose Verification dialog box opens.
  2. Select the Dynamic Verification radio button and click OK.
    The IDE starts the instrumentation of the application server instance.
