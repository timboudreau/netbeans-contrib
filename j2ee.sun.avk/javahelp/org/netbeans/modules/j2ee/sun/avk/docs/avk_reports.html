Generating and Viewing AVK Reports

See Also

Note : This section is applicable only if your project is using Sun Java System Application Server 8.x as runtime.

Before you can generate and view an Application Verification Kit (AVK) report, you must enable dynamic verification and manually execute a sample.

To generate and view an AVK report:

  1. In the Runtime window, expand the Servers node.
  2. Right-click the Sun Java System Application Server node and choose Generate AVK Report.
  3. At the Warning dialog box, click OK to stop the server, stop AVK instrumentation, and generate the AVK report.
    A browser window opens with the Java AVK Report.
  • The generated report and the result files are stored in the ide-user-directory/avkreport/results directory.
